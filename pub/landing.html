MyDoodles.js

This is the random theme application in block with a message, where user can refresh the block and the background color will randomly change to a color. There are also function that allows user to select the color, which are presented below. Sample use case are any website that want automatically generated background color, or some website that a "sentence for today" etc.

This is the weather theme application in block, where in the real world cases the weather data and image are get from server. Sample use

This is the calender + mood theme application in block, where there is an user interaction for user to choose their mood for that day. Once the mood were chosen, the corresponding words for moods will presented and the background color of webpage will be changed. This can be used for website like online website for diary where different background can be presented with different selection for that day.
